
# LIV ZONE - Fast Food and Bar
 ## Informatii Generale Proiect
Acest website reprezintă produsul final pentru participarea la Concursul Tekwill Junior Ambassadors, Ediția a VI-a. Proiectul este încadrat la categoria Proiectare și Dezvoltare Web și este destinat unei afaceri reale din comunitatea locală.

## Obiectivul Principal
Digitalizarea prezenței online a localului LIV ZONE (COTOROBAI FAMILY S.R.L.), situat în Răzeni, pentru a oferi clienților o modalitate modernă de a consulta meniul și informațiile de contact.

## Echipa si Institutia
Proiectul a fost dezvoltat în cadrul instituției de învățământ, respectând criteriile de colaborare și mentorat impuse de regulament.

# Detalii Tehnice si Implementare
Site-ul a fost construit folosind tehnologii web standard, punând accent pe scrierea manuală a codului pentru a demonstra competențele tehnice dobândite.

## Tehnologii Utilizate
HTML5: Utilizat pentru structurarea logică și semantică a fiecărei pagini.

CSS3: Implementat pentru design, layout-uri de tip responsive și stilizarea componentelor vizuale.

JavaScript: Folosit pentru funcționalitățile interactive ale interfeței.

## Criterii de Evaluare si Punctaj
Conform Anexei 2 din regulament, proiectul a fost optimizat pentru a acoperi toate ariile de evaluare.

### Design si UI/UX
S-a urmărit crearea unei interfețe intuitive care să asigure o experiență fluidă pe orice dispozitiv (Desktop, Tabletă, Mobil). Navigarea este unitară, iar identitatea vizuală respectă specificul unui local de tip Fast Food and Bar.

### Interactivitate JavaScript
Codul JavaScript gestionează elemente precum meniul adaptiv pentru mobil și animațiile care îmbunătățesc prezentarea informațiilor, respectând cerințele de dinamism ale paginii.

### Aspecte Tehnice si Validitate
Codul este organizat, comentat și respectă bunele practici de dezvoltare. S-au utilizat meta-tag-uri pentru responsivitate și SEO, asigurând o indexare corectă și o afișare optimă.

### Structura Fisierelor din Proiect
Organizarea fișierelor reflectă o structură profesională de dezvoltare web.

### Pagini Principale
index.html: Pagina de pornire cu informațiile de bază.

about.html: Detalii despre istoria și echipa localului.

products.html: Meniul digital interactiv.

contact.html: Formular de contact și localizare geografică.

### Resurse si Logica
styles.css: Fișierul de stilizare generală.

main.js: Logica pentru interactivitate.

images/: Folder dedicat resurselor media optimizate.

## Impactul asupra Comunitatii
Prin crearea acestui site, afacerea locală LIV ZONE beneficiază de o vizibilitate sporită în mediul digital. Proiectul demonstrează cum tehnologia poate sprijini antreprenoriatul local din Răzeni, oferind un instrument util atât proprietarilor, cât și clienților.
